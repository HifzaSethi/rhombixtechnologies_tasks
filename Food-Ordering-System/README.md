# 🍔 Food Ordering System – Full Stack Application

## 🚀 Overview

A full-stack food ordering platform built using:

- **Next.js (Frontend)**
- **Express.js (Backend API)**
- **MongoDB (Database)**
- **JWT Authentication**
- **Mongoose ODM**

This system allows:

✅ Admin to manage restaurants & menu
✅ Users to add items to cart
✅ Users to place orders
✅ Admin to manage orders

## 🏗 Architecture

Frontend (Next.js)
↓
API Calls (Axios)
↓
Backend (Express.js + JWT Middleware)
↓
MongoDB Database

## 🔐 Authentication

- User Register/Login
- Password hashed using bcrypt
- JWT token stored in localStorage
- Protected routes for:
  - Cart
  - Orders
  - Admin Dashboard

## 👑 Admin Features

- Add / Delete Restaurants
- Add / Delete Menu Items
- View All Restaurants & Menu
- Manage Orders

## 👤 User Features

- Browse Restaurants
- View Menu
- Add Items to Cart
- Remove Items from Cart
- Place Orders
- View Order Status
